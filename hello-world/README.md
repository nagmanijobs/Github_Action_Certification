# Hello World (sample)

This directory contains a minimal HTML/CSS/JS "Hello World" sample app.

Try it locally:

1. Open `hello-world/index.html` in your browser (double-click).
2. Or serve with a simple HTTP server and visit the URL below:

```sh
python3 -m http.server 8000
# then open http://localhost:8000/hello-world/
```
